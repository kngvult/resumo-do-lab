# resumo-do-lab
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO

O que aprendi
Durante o desenvolvimento do lab na DIO, explorei os principais conceitos do Microsoft Azure e como localizar serviços por categoria. Entre os pontos mais importantes:

Organização dos Serviços: O Azure possui centenas de serviços, mas todos são agrupados em categorias como Computação, Rede, Armazenamento, Banco de Dados, Segurança, Inteligência Artificial, DevOps, entre outros.
Portal do Azure: Aprendi a navegar pelo portal e identificar rapidamente os serviços disponíveis em cada categoria.

Exemplos de Serviços:
Computação: Máquinas Virtuais, App Services, Functions.
Armazenamento: Blob Storage, Disk Storage, File Storage.
Banco de Dados: Azure SQL Database, Cosmos DB, MySQL.
Rede: Virtual Network, Load Balancer, VPN Gateway.
Importância da Categorização: Facilita a escolha do serviço certo para cada necessidade, evitando confusões e otimizando o tempo de configuração.
Prática no Lab: Criei recursos no portal, explorei diferentes categorias e entendi como cada serviço pode ser aplicado em  cenários reais.

Conclusão
Esse lab me ajudou a compreender melhor a estrutura do Azure e como localizar serviços de forma eficiente. Agora consigo identificar rapidamente onde cada recurso se encaixa e como utilizá-lo em projetos futuros.
